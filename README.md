# GeoSAM

**Segment Anything Model for seismic interpretation.**

GeoSAM applies Meta's SAM2 foundation model to seismic data — generating
geological feature masks without any training data required.

## Install

```bash
pip install geosam
pip install geosam[sam]
pip install geosam[all]
```
